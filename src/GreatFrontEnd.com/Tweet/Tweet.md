Requirements

- Tweet
  - Background color: #fff
  - Border: 1px #cfd9de
  - Border radius: 16px
  - Padding: 12px
- Thumbnail
  - Size: 48px
  - Fully rounded
- Name
  - Color: #000
  - Font: sans-serif 15px bold
- Metadata
  - Font: #71767b sans-serif 15px
  - Use &middot; for the dot
- Message
  - Color: #0f1419
  - Font family: sans-serif 15px
- Actions Row
  - Max width: 425px
  - Equal spacing between each action
- Icon
  - Color: #536471
  - Size: 16px
  - Vertically centered with the count
- Action Count
  - Color: #536471
  - Font size: 13px
